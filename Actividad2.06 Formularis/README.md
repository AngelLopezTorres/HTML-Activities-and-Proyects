Crea un document HTML que contingui un formulari de registre d'una pàgina web de
videojocs on se sol·licita els camps següents:
  1. Nom
  2. Edat
  3. Correu electrònic
  4. Contrasenya
  5. Un selector amb grups per triar una o més consoles favorites, entre les següents: Consoles de casa i Consoles Portàtils .
      Per a una millor visualització heu de visualitzar el selector amb un nombre de línies de 9.
  7. Un radio button per poder seleccionar un únic idioma entre castellà, català o
anglès.
  8. Una casella perquè l'usuari es pugui subscriure o no a les notificacions per
correu electrònic
  9. Un camp per a comentaris.
  10. Un botó per enviar la informació del formulari.
