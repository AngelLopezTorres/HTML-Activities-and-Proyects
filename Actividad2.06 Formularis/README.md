Crea un document HTML que contingui un formulari de registre d'una pàgina web de
videojocs on se sol·licita els camps següents:
  1. Nom
  2. Edat
  3. Correu electrònic
  4. Contrasenya
  5. Un selector amb grups per triar una o més consoles favorites, entre les següents: Consoles de casa i Consoles Portàtils .
      Per a una millor visualització heu de visualitzar el selector amb un nombre de línies de 9.
  7. Un radio button per poder seleccionar un únic idioma entre castellà, català o
anglès.
  8. Una casella perquè l'usuari es pugui subscriure o no a les notificacions per
correu electrònic
  9. Un camp per a comentaris.
  10. Un botó per enviar la informació del formulari.
  S’han de complir les següents condicions i restriccions:
    ● Tots els camps han de tenir la seva etiqueta, identificant a l’usuari, de que es
      tracta el camp.
    ● Els camps de nom, edat, correu electrònic i contrasenya són obligatoris.
    ● Només es poden registrar adults, així que l’edat ha de ser mínim de 18 anys.
    ● La contrasenya ha de tenir una longitud mínima de 6 caràcters y un màxim de
    12. A més, no es poden anar visualitzant els caràcters, segons l'usuari estigui
    escrivint.
    ● S’han de poder seleccionar una o més consoles com a favorites.
    ● El camp de comentaris ha de tenir una amplada de 50 columnes i un nombre
    de línies de 7.
    ● En prémer el botó, el formulari ha d'obrir un nou HTML indicat a l'atribut
    action. Aquest nou HTML només ha de tenir un títol i el text "Gràcies per
    registrar-te!".
    ● Els dos HTML han d'estar al mateix directori.
