Crea un nou HTML per mostrar el teu portfoli personal amb les condicions següents:
S'ha de crear mitjançant una estructura completa d'HTML.
Ha de tenir una capçalera amb el títol i els metadades bàsics.
El body ha de tenir les següents seccions diferències:
Informació personal, amb el nom i una descripció breu
Estudis, amb una llista de tots els estudis oficials i cursos. Marca en un color diferent el darrer estudi realitzat.
Llenguatges de programació, amb la llista dels llenguatges de programació que conec.
Peu amb informació del copyright, com a exemple "Copyright © 2025 Ángel Cristian López Torres".
